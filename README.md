# Kalkulator Średniej Ocen - Android

Aplikacja mobilna na system Android do obliczania średniej ocen z przedmiotów informatycznych.

## Funkcjonalności

- Wprowadzenie imienia, nazwiska i liczby przedmiotów (5–15)
- Wybór oceny (2–5) dla każdego przedmiotu za pomocą przycisków radiowych
- Automatyczne obliczanie średniej po zaznaczeniu wszystkich ocen
- Walidacja pól formularza w czasie rzeczywistym
- Obsługa obrotu ekranu bez utraty danych

## Przedmioty

Aplikacja obsługuje 15 przedmiotów informatycznych


## Technologie

- Java
- Android SDK

## Uruchomienie

1. Sklonuj repozytorium
2. Otwórz projekt w Android Studio
3. Uruchom na emulatorze lub urządzeniu fizycznym
